对Vue.js需要注意的地方进行了归纳和整理

vue基础

用户会看到花括号标记

v-cloak + [v-cloak] {display: none;}  // 阻止闪烁

v-text="msg" // 同样可以阻止闪烁

外部挂载vue对象 ==> $mount(‘#box’)

var vm = new Vue({
    data:{}
}).$mount('#box');

vue动画效果

  • 在DOM节点中加入<transition name="fade">...</transition> 标签对，将需要设置动画的节点包裹起来
  • css3实现动画效果
<style>
  .fade-enter-active,
  .fade-leave-active {
    transition: all 1s ease;   //所有的动画效果在1s内完成，淡入淡出
  }

  .fade-enter,
  .fade-leave-to {
    opacity: 0;  //透明度
    transform: translateX(200px);  //沿X轴位移200px
  }
<style>
  • 结合animate.css
//先引入bower安装的animate.css文件
<link rel="stylesheet" href="bower_components/animate.css/animate.css">

//在节点中应用
<transition enter-active-class="animeted zoomInLeft" leave-active-class="animated zoomOutRight">
  ...
</transition>

vue中的组件

  • 在Vue.component中加入data 数据的时候需要写成函数形式
Vue.component('...', {
  data() {
    return { //返回的必须是个json对象
      msg: ...
    }
  },
  template: `...`
});
  • 组件写在<template id="..."></template> 内部
  • 动态组件：<div :is="..."></div> is后面绑定的组件是什么显示的就是什么组件

子组件调用父组件的data

  • 在子组件中如果需要调用父组件内的data ，需要在dom节点上:msg="fatherMsg" 绑定元素，然后再在component 内用props 获取元素
  • props 有两种写法：
//数组写法
props: ['msg1', 'msg2']

//对象的写法
props: {
  'msg1': String,  //后面是数据类型
  'msg2': Number
}

父组件调用子组件的数据

  • 利用this.$emit('event_name', msg) 向父组件传递事件，父组件通过@event_name="method_name" 接收事件后调用methods 方法对传入的msg 进行处理

创建Vue标准模板

  • 安装淘宝镜像： npm install -g cnpm --registry=https://registry.npm.taobao.org
  • cnpm install vue-cli -g 全局安装vue-cli 脚手架`
  • vue init webpack <project_name> 初始化vue标准模板
  • npm run dev 运行程序，npm run build 开始打包文件
  • 可以在./config/index.js 文件中修改配置选项，包括端口号、是否自动打开浏览器

vue-router 路由

  • 在初始化Vue项目的时候可以选择是否安装router
  • 在src目录下新建router文件夹，创建index.js作为路由入口
  • router目录的index.js中写入如下代码
import Vue from 'vue'
import VueRouter from 'vue-router'
import Index from '../components/Index'  // 路由跳转的目录

Vue.use(VueRouter); // 注册插件

export default new VueRouter({
  routes: [{
    path: '/',
    component: Index,
    alias: '/index', //别名
    redirect: '/register' //重定向 -> 当点击主页的时候会自动跳转到注册页
  }]
});
  • 在main.js中加入import router from './router' ，Vue对象中加入router
  • 根据需要在components目录下创建<page_name>.vue

Vuex

  • 安装Vuex ， cnpm install vuex -S ，-S的作用是保存配置
  • 引入方式和vue-router一样，在main.js中通过import引入
  • 新建./store 目录，在其中创建index.js 文件
import Vue from 'vue';
import Vuex from 'vuex';
import createLogger from 'vuex/dist/logger' //生成日志

Vue.use(Vuex); // 注册插件    install方法

export default new Vuex.Store({

  plugins: [createLogger()], //插件 -> 生成log日志

  state: {   // 存放数据
    x: 1  
  },
  mutations: {  // 唯一改变state的方法（通常情况下state中的数据是只读的）
    increment(state) {
      state.x++
    }
  }
});
  • 通过import <plugin_name> from 'vuex/dist/...' 来引入插件,然后再通过plugins: [<plugin_name>()] 的方式使用
  • 当.vue文件想要调用vuex中数据的时候，可以通过函数返回return this.$store.state.<x> 来获取
  • 调用vuex中的方法时可以使用this.$store.commit('<methods_name>')
